## 📋 模組概述

Climate Module 提供氣候與環境監測功能，整合氣象資料、環境感測器，協助工地施工決策與安全管理。

### 業務範圍

- 天氣預報查詢與警報
- 環境感測器資料收集
- 氣候條件分析
- 施工建議與警告

### 核心特性

- ✅ **天氣預報**: 整合第三方氣象 API
- ✅ **環境監測**: 溫度、濕度、風速等感測器資料
- ✅ **警報通知**: 惡劣天氣自動警報
- ✅ **歷史資料**: 氣候資料儲存與分析

---

## 📦 核心功能

### 天氣預報
整合 OpenWeather API 提供即時與未來天氣預報。

### 環境感測器
支援 IoT 感測器資料收集與即時監控。

### 警報系統
惡劣天氣自動發送警報通知相關人員。

### 資料分析
氣候資料統計與趨勢分析。
