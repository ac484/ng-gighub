## 📋 模組概述

Cloud Storage Module 提供統一的雲端儲存介面，支援檔案上傳、下載、管理等功能。整合 Firebase Storage 提供安全可靠的檔案儲存服務。

### 業務範圍

- 檔案上傳與下載
- 檔案管理與組織
- 存取權限控制
- 檔案分享與連結

### 核心特性

- ✅ **統一儲存介面**: 抽象雲端儲存服務
- ✅ **安全上傳**: 檔案類型與大小限制
- ✅ **權限控制**: 細粒度存取權限管理
- ✅ **分享功能**: 臨時分享連結生成

---

## 📦 核心功能

### 檔案上傳
支援單檔案與多檔案上傳，自動壓縮與格式轉換。

### 檔案管理
檔案列表、搜尋、重新命名、刪除等管理功能。

### 存取控制
基於角色的存取控制（RBAC）。

### 檔案分享
臨時分享連結，支援密碼保護與有效期限。
