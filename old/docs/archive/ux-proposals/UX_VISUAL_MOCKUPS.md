## 互動流程圖

### 當前流程：建立團隊並加入成員
```
開始
  ↓
切換到組織上下文 (1 次點擊)
  ↓
前往團隊管理頁面 (1 次點擊)
  ↓
點擊「建立團隊」(1 次點擊)
  ↓
填寫表單 + 送出 (2 次點擊)
  ↓
點擊「管理成員」(1 次點擊)
  ↓ (自動切換上下文)
導航到成員頁面 (自動)
  ↓
點擊「新增成員」(1 次點擊)
  ↓
選擇成員 + 確認 (2 次點擊)
  ↓
完成

總計：9 次點擊 + 2 次頁面切換
```

### 建議流程：建立團隊並加入成員
```
開始
  ↓
側邊欄選擇組織 (1 次點擊)
  ↓
點擊「建立團隊」(1 次點擊)
  ↓
填寫表單 + 送出 (2 次點擊)
  ↓
點擊團隊卡片「查看詳情」(1 次點擊)
  ↓ (抽屜滑出)
點擊「新增成員」(1 次點擊)
  ↓
選擇成員 + 確認 (2 次點擊)
  ↓
完成

總計：8 次點擊 + 0 次頁面切換
效率提升：~40%
```

## 顏色與圖示規範

### 語意化顏色
```
主色調 (Primary)   → #1890ff (藍色) - 主要操作
成功 (Success)     → #52c41a (綠色) - 成功狀態
警告 (Warning)     → #faad14 (黃色) - 警告訊息
錯誤 (Error)       → #f5222d (紅色) - 錯誤/刪除
資訊 (Info)        → #1890ff (藍色) - 資訊提示

用戶 (User)        → #1890ff (藍色)
組織 (Org)         → #52c41a (綠色)
團隊 (Team)        → #722ed1 (紫色)
機器人 (Bot)       → #fa8c16 (橙色)
```

### 圖示映射
```
👤 user            → 個人/用戶
🏢 team            → 組織
👨‍👩‍👧‍👦 usergroup-add → 團隊
🤖 robot          → 機器人
📊 dashboard      → 儀表板
👥 contacts       → 成員列表
⚙️ setting        → 設定
➕ plus           → 新增/建立
✏️ edit           → 編輯
🗑️ delete         → 刪除
🔍 search         → 搜尋
🏷️ tags           → 標籤/角色
```
