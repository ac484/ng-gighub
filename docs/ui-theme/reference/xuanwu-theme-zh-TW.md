- 顏色: 深藍、暗藍、青綠、銀灰、近黑

## 🌈 漸層系統

### 主要漸層

#### 1. 北方之水
```css
background: linear-gradient(135deg, #1E3A8A 0%, #0D9488 100%);
```
**用途**: 主要按鈕、主視覺區、特色卡片  
**效果**: 從深藍到青綠的對角漸層  
**象徵**: 永恆流動的北方之水

#### 2. 龜甲紋理
```css
background: linear-gradient(45deg, #172554 0%, #1E3A8A 50%, #64748B 100%);
```
**用途**: 裝飾圖案、背景  
**效果**: 多點角度漸層  
**象徵**: 防護龜甲的遠古紋理

#### 3. 冬夜深沉
```css
background: linear-gradient(180deg, #0F172A 0%, #1E3A8A 50%, #0D9488 100%);
```
**用途**: 大型橫幅、頁面背景  
**效果**: 垂直深到淺漸層  
**象徵**: 夜空過渡到北方水域

#### 4. 銀霜微光
```css
background: linear-gradient(135deg, #EFF6FF 0%, #E2E8F0 50%, #CBD5E1 100%);
```
**用途**: 表頭、卡片背景、細微高亮  
**效果**: 溫和冷色調漸層  
**象徵**: 冬霜與晨冰

#### 5. 深淵神秘
```css
background: linear-gradient(135deg, #172554 0%, #115E59 100%);
```
**用途**: 暗色模式、夜間主題元素  
**效果**: 深藍到暗青綠  
**象徵**: 深不可測的深淵

## 📐 使用指南

### 建議做法 ✅

1. **使用主色進行操作**
   ```html
   <button nz-button nzType="primary">提交</button>
   ```

2. **一致使用語義色**
   ```html
   <nz-alert nzType="success" nzMessage="成功!"></nz-alert>
   <nz-alert nzType="error" nzMessage="錯誤!"></nz-alert>
   ```

3. **為主視覺元素使用漸層**
   ```html
   <div class="hero-banner" style="background: linear-gradient(135deg, #1E3A8A 0%, #0D9488 100%)">
     <h1>歡迎來到 GigHub</h1>
   </div>
   ```

4. **使用陰影增加深度**
   ```less
   .card {
     box-shadow: @shadow-xuanwu-md;
     &:hover {
       box-shadow: @shadow-xuanwu-lg;
     }
   }
   ```

### 應避免 ❌

1. **不使用純黑**
   - ❌ `color: #000000`
   - ✅ `color: #0F172A` (Slate 900)

2. **不過度使用漸層**
   - ❌ 每個元素都有漸層
   - ✅ 保留給重要元素

3. **不忽略對比度**
   - ❌ xuanwu-4 文字在白背景上
   - ✅ xuanwu-8 或更深用於文字

4. **不混合太多顏色**
   - ❌ 同時使用所有漸層
   - ✅ 每個視圖選擇 1-2 個漸層

## 🔧 實作方法

### 方法 1: 執行時配置 (app.config.ts)

```typescript
import { NzConfig, provideNzConfig } from 'ng-zorro-antd/core/config';

const ngZorroConfig: NzConfig = {
  theme: {
    primaryColor: '#1E3A8A',
    successColor: '#0D9488',
    warningColor: '#F59E0B',
    errorColor: '#EF4444',
    infoColor: '#64748B'
  }
};

export const appConfig: ApplicationConfig = {
  providers: [provideNzConfig(ngZorroConfig)]
};
```

### 方法 2: 編譯時 Less 變數 (theme.less)

```less
// 在導入 @delon/theme 之前定義
@primary-color: #1E3A8A;
@success-color: #0D9488;
@warning-color: #F59E0B;
@error-color: #EF4444;
@info-color: #64748B;

@import '@delon/theme/theme-default.less';
```

### 方法 3: 動態主題切換

```typescript
import { NzConfigService } from 'ng-zorro-antd/core/config';
import { inject } from '@angular/core';

export class ThemeService {
  private nzConfigService = inject(NzConfigService);
  
  changeToXuanwuTheme() {
    this.nzConfigService.set('theme', {
      primaryColor: '#1E3A8A'
    });
  }
  
  changeToCustomTheme(color: string) {
    this.nzConfigService.set('theme', {
      primaryColor: color
    });
  }
}
```

## 🌍 無障礙性

### WCAG 2.1 合規

所有色彩組合達到 **WCAG 2.1 Level AA** 標準:

- ✅ 主文字 (`#1E293B`) 在白色上: **14.8:1** (AAA)
- ✅ Xuanwu-6 (`#1E3A8A`) 在白色上: **8.9:1** (AAA)
- ✅ Teal-4 (`#0D9488`) 在白色上: **4.5:1** (AA)
- ✅ 連結色在白色上: **8.9:1** (AAA)
- ✅ 次要文字 (`#64748B`) 在白色上: **5.3:1** (AA)

### 色盲支援

- ✅ 紅綠色盲: 使用不同色相 (藍 vs. 紅)
- ✅ 藍黃色盲: 維持足夠對比
- ✅ 單色: 適當的明度差異

### 額外指標

絕不僅依賴色彩:
- ✅ 狀態圖示 (✓, ✗, ⚠, ℹ)
- ✅ 狀態文字標籤
- ✅ 必要時的圖案或紋理
