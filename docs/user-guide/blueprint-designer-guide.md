# Blueprint Designer 使用指南

> **更新日期**: 2025-12-11  
> **版本**: v2.0 (含連接系統與驗證)

## 快速開始

Blueprint Designer 提供視覺化的模組配置介面。

### 主要功能

1. **拖拽模組**: 從左側選擇器拖拽模組到畫布
2. **建立連接**: 拖拽連接端點建立模組依賴關係
3. **自動驗證**: 系統自動檢測循環依賴等錯誤
4. **視覺化回饋**: 即時顯示配置狀態

### 建立連接

1. 按住來源模組右側的藍色輸出端點
2. 拖拽到目標模組左側的藍色輸入端點
3. 放開滑鼠完成連接

### 驗證錯誤

系統會自動檢測:
- 循環依賴: A → B → C → A
- 缺失模組: 連接指向不存在的模組
- 無效連接: 模組連接到自己

修正所有錯誤後才能儲存藍圖。
